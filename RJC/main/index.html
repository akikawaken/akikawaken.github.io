Json構文エラーとおさらばしましょう。

RtmJsonCreatorがあなたのアドオン制作を支援します。

インストール
For Windows
curl -sLJO https://github.com/akikawaken/Rtm_Json_Creator/releases/download/update/RtmJsonCreator.bat & call RtmJsonCreator.bat
または、GitHubからダウンロード。
上記コマンドを利用してインストールすると自動で初期設定が走り、利用開始が可能です。

MacまたはLinuxをお使いですか?
GUI版が利用可能です。
(GUI版は開発途中の為、CLI版の一部機能のみ利用可能です)


RtmJsonCreatorとは?

RtmJsonCreatorはRTMのアドオン用Json作成支援ツールです。
CLI版とGUI版の二種類があります。

コマンドライン版

<-バナークリックでGitHubにリンクします。

利用可能なJsonタイプ

- 列車
- 看板
- 機能あり設置物
- 機能なし設置物
- NPC
- 旗
- 信号機
- レール
- コンテナ
- 火器
- コネクタ
- ワイヤ
- 乗り物(自動車,航空機,船舶,リフト)
- sounds.json
- pack.json

RtmJsonCreator(CLI版)の特徴

- 値を入力するだけなので Json構文エラー の可能性が低くなる
- キーボードだけで操作でき、普段のコマンドライン操作と同じ感覚で利用できる (一部は補完が利用可能!)
- オフライン環境でも実行できる
- MITLicenseで利用できる
- MQOファイルパスを入力するだけで自動でモデルファイル設定を完了してくれる、自動材質設定が利用できる

動作環境

10,11のWindowsならほぼ確で動く。
7は(RJCの)互換モードでぎりいけるか? って感じ。 xpまで行くと無理だと思う

使い方

バッチファイルを起動し、表示された内容に従って入力するだけ。
