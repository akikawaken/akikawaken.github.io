看板の json を作成します


























これは看板の発光の度合いを設定します。
-16 ランダムに点滅する
-15~-1 RS 信号が入っている時、発光する(数字は発光の強さ)
0 発光しない
1~15 常に発光する(数字は発光の強さ)


------ファイル名:signboard_.json-------
